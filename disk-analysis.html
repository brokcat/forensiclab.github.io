Disk Analysis Fundamentals

Learn essential techniques for analyzing disk images, recovering deleted files, and investigating filesystem artifacts.

Introduction to Disk Analysis

Disk analysis is a critical component of digital forensics that involves examining storage media to recover evidence, analyze file systems, and reconstruct user activities. This course covers fundamental techniques and tools used in disk forensics investigations.

File System Basics

Understanding file systems is essential for effective disk analysis. We'll explore common file systems like NTFS, FAT, ext4, and their key structures:

  • Master File Table (MFT): The database that tracks all files on an NTFS volume
  • Journal: Records changes to the file system metadata
  • File Allocation Table: Maps clusters to files in FAT file systems
  • Inodes: Data structures in Unix/Linux file systems that store file metadata

Deleted File Recovery

When files are deleted, they aren't immediately removed from the disk. Instead:

  1. The file system marks the space as available for new data
  2. File entries may be modified or removed from directory listings
  3. The actual data remains until overwritten

This course will teach you techniques to recover these deleted files and analyze their contents.
