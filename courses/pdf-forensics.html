1. Structure des fichiers PDF

Complété

Les fichiers PDF (Portable Document Format) sont des documents complexes avec une structure interne spécifique qui peut contenir divers types de contenu, y compris du texte, des images, des formulaires, des scripts et des métadonnées.

Composants clés d'un fichier PDF:
  • En-tête - Indique la version du format PDF
  • Objets - Éléments de base qui composent le document (texte, images, etc.)
  • Table de références croisées (xref) - Index des objets dans le fichier
  • Trailer - Contient des pointeurs vers des structures importantes
  • Dictionnaire de catalogue - Structure principale du document
%PDF-1.7    # En-tête
1 0 obj     # Objet numéro 1
<< /Type /Catalog /Pages 2 0 R >>
endobj

2 0 obj     # Objet numéro 2
<< /Type /Pages /Kids [3 0 R] /Count 1 >>
endobj

xref        # Table de références croisées
0 4
0000000000 65535 f
0000000010 00000 n
0000000079 00000 n

trailer     # Trailer
<< /Size 4 /Root 1 0 R >>
startxref
183
%%EOF
Point clé: La structure flexible des PDF permet de cacher du contenu malveillant de différentes manières, ce qui en fait un format souvent utilisé pour les attaques.

2. Extraction de métadonnées

Complété

Les métadonnées des fichiers PDF peuvent révéler des informations précieuses sur l'origine du document, son auteur, les logiciels utilisés pour le créer et les modifications apportées.

Outils d'extraction de métadonnées:
Outil Type Description
ExifTool Open Source Outil polyvalent pour extraire les métadonnées de divers formats
pdfinfo Open Source Partie de la suite Poppler, affiche les informations de base
peepdf Open Source Outil d'analyse forensique spécifique aux PDF
PDF Stream Dumper Gratuit Outil d'analyse avancé pour Windows
Commandes d'extraction de métadonnées:
# Utilisation d'ExifTool
exiftool document.pdf

# Utilisation de pdfinfo
pdfinfo document.pdf

# Utilisation de peepdf
peepdf -i document.pdf
Conseil: Comparez les métadonnées extraites par différents outils, car certains peuvent révéler des informations que d'autres manquent.
