1. Introduction à l'analyse de disque

Complété

L'analyse de disque forensique est une technique fondamentale en investigation numérique qui permet d'examiner le contenu des supports de stockage tels que les disques durs, SSD, clés USB et cartes mémoire.

Pourquoi l'analyse de disque est importante:
  • Récupération de fichiers supprimés ou cachés
  • Analyse de la chronologie des activités sur un système
  • Identification de preuves de compromission
  • Extraction de données utilisateur et de configuration système
  • Détection de techniques anti-forensiques
Point clé: Toujours travailler sur une copie du disque original pour préserver l'intégrité des preuves.

2. Acquisition d'images disque

Complété

L'acquisition d'une image disque est la première étape cruciale de l'analyse forensique. Elle consiste à créer une copie bit à bit exacte du support de stockage original.

Outils d'acquisition d'images disque:
Outil Type Description
FTK Imager Commercial/Gratuit Outil polyvalent pour créer des images disque
DD / DCFldd Open Source Utilitaires en ligne de commande Linux
Guymager Open Source Interface graphique pour Linux
EnCase Commercial Suite forensique professionnelle
Attention: Toujours utiliser des bloqueurs en écriture matériels ou logiciels lors de l'acquisition pour éviter de modifier les preuves.
